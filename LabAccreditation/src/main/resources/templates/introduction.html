1. ทำไมต้อง ISO/IEC 17025

การแข่งขันทางการค้า และการดำเนินธุรกิจของภาคอุตสาหกรรมในปัจจุบันมีความจำเป็นอย่างยิ่ง ที่ต้องให้ความสำคัญกับคุณภาพและมาตรฐาน เพื่อให้สินค้าเป็นที่ยอมรับอย่างเป็นสากล ผู้ผลิตสินค้า ต้องได้รับการรับรองระบบคุณภาพในส่วนที่เกี่ยวข้องกับการผลิต การบริหารงาน และการบริการ เช่น ISO 9000, ISO 14000, ISO 18000 ฯลฯ รวมทั้งสินค้าต้องผ่านการตรวจสอบคุณภาพว่า เป็นไปตามมาตรฐานสินค้า ดังนั้น เพื่อแสดงให้เห็นถึงกระบวนการควบคุมคุณภาพสินค้าอย่างมีมาตรฐาน ผลการตรวจสอบสินค้าจึงต้องมาจากห้องปฏิบัติการทดสอบที่ได้มาตรฐาน ISO/IEC 17025 ที่ได้ผลการทดสอบมีความน่าเชื่อถือ ทำให้ผู้บริโภคมั่นใจว่าได้รับสินค้าที่ผ่านกรรมวิธีการผลิตและการตรวจสอบที่มีคุณภาพ

2. ใครควรทำ

ภาคอุตสาหกรรมการผลิตที่มีห้องปฏิบัติการตรวจสอบคุณภาพสินค้า หรือห้องปฏิบัติการที่ให้บริการทดสอบสินค้าหรือสอบเทียบเครื่องมือทดสอบ ควรได้การรับรอง ISO/IEC 17025 เพื่อให้เกิดความมั่นใจคุณภาพและความน่าเชื่อถือ ลดการกีดกันทางการค้าและลดการตรวจซ้ำจากประเทศคู่ค้า

3. ทำอย่างไร

ห้องปฏิบัติการที่ต้องการขอการรับรอง ISO/IEC 17025 สามารถสำรวจความพร้อม เพื่อเตรียมยื่นขอการรับรอง ได้บนเว็บไซต์

4. ขอที่ไหน

สามารถยื่นขอการรับรอง ได้จากหน่วยรับรองในประเทศไทย ซึ่งมี 3 หน่วยงาน คือ

  1. กรมวิทยาศาสตร์บริการ : ให้การรับรองห้องปฏิบัติการทดสอบ
  2. สำนักงานมาตรฐานผลิตภัณฑ์อุตสาหกรรม : ให้การรับรองห้องปฏิบัติการทดสอบและสอบเทียบ
  3. กรมวิทยาศาสตร์การแพทย์ : ให้การรับรองห้องปฏิบัติการทดสอบด้านอาหาร และทางการแพทย์

โดยทั้ง 3 หน่วยงาน ได้รับการยอมรับในด้านความสามารถทางวิชาการทั้งในระดับภูมิภาค และระดับโลก ในกลุ่มประเทศสมาชิก APLAC และ ILAC

5. เตรียมอย่างไร

  • กำหนดขอบข่ายการทดสอบ ตามความจำเป็น ตามความต้องการของลูกค้า และศักยภาพของห้องปฏิบัติการ
  • ศึกษาข้อกำหนด ISO/IEC 17025 และข้อกำหนดที่เกี่ยวข้อง
  • ผู้บริหารให้ความเห็นชอบในการจัดทำระบบคุณภาพของห้องปฏิบัติการ
  • กำหนดหน้าที่ความรับผิดชอบของแต่ละตำแหน่ง (job description)
  • จัดทำแผนอบรม ให้ความรู้ อบรมบุคลากรที่เกี่ยวข้อง ทั้งด้านการบริหารและด้านวิชาการ
  • จัดทำเอกสารคู่มือคุณภาพ ขั้นตอนการดำเนินงาน วิธีปฏิบัติงาน วิธีทดสอบ และเอกสารสนับสนุนที่จำเป็น
  • ประกาศใช้เอกสารในระบบบริหารงานคุณภาพ
  • ดำเนินการด้านวิชาการต่างๆ การทดสอบ เครื่องมือ อุปกรณ์ สารเคมี วัสดุอ้างอิงรับรอง/วัสดุอ้างอิง ขั้นตอนการจัดการตัวอย่าง การประกันคุณภาพผลการดสอบ การออกรายงานผลการทดสอบ/li>
  • มอบหมายงานแก่บุคลากร
  • ดำเนินการตามระบบคุณภาพให้ครบทุกกิจกรรม และครบทั้งกระบวนการตั้งแต่การรับตัวอย่าง การทดสอบ การควบคุมคุณภาพภายใน/ภายนอก และการออกรายงานผลการทดสอบ
  • ติดต่อหน่วยรับรองระบบงานห้องปฏิบัติการ เพื่อยื่นขอการรับรอง
